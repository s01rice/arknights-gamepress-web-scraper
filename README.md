# arknights-gamepress-web-scraper
A basic script that akes operator data from arknights.gamepress.gg.

By default, script is set to only take operators available in NA.

The basic structure:

- Operator Name
- Operator Icon
- Operator Stats (max trust):
    - Rarity
    - HP
    - ATK
    - DEF
    - Cost
    - RES
    - Block count
    - Attack interval
- Traits
- Talents []:
    - Name
    - Description
- Skills []:
    - Name
    - Icon
    - Description
    - SP Cost
    - Initial SP
    - Charge type
    - Activation type
    - Duration
